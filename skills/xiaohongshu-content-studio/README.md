## 1. 项目概述

本仓库是一个 **Codex / Claude Code 双平台通用 Skill**，用于生成本地化、真实感强的小红书内容包——包括标题、封面文字、正文、轮播图方向和发布策略。

## 2. 目标与场景

适用场景：

- 个人品牌内容策划与输出。
- 项目/产品分享的社交平台包装。
- AI 工具使用经验分享。
- 学习笔记、前后对比类内容。
- 需要平台原生感而非通用 AI 写作风格的场景。

## 3. 核心能力

- 多种钩子风格的标题生成。
- 封面文字与视觉方向建议。
- 小红书节奏感的正文撰写。
- 轮播图/配图内容方向规划。
- 完整的发布内容包输出。

## 4. Quick Start

**Codex**：在 skill 市场中搜索 `xiaohongshu-content-studio` 一键安装。

**Claude Code**：将本仓库克隆到 skills 目录或通过 plugin 加载：

    git clone https://github.com/however-yir/codex-skill-xiaohongshu-content-studio.git
