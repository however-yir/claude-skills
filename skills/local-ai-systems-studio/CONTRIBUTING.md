# Contributing

Thanks for your interest in improving this skill.

## Good Contributions

- clarify trigger conditions
- improve examples or decision criteria
- tighten stack-selection guidance
- expand references that help people choose practical local setups
- improve the handoff boundary with related workflow skills

## Please Keep Intact

- the distinction between local stack choice and deeper workflow orchestration
- the honesty of deployment tradeoffs and examples
- the lightweight repository structure

## When You Change Behavior

If you change what the skill is supposed to do, please update all of the following:
- `SKILL.md`
- `README.md`
- `examples/showcase.md`
- any affected reference files

## Style Notes

- prefer additive improvements over unnecessary rewrites
- keep examples concrete and realistic
- avoid overcomplicating the recommended local stack unless it solves a real problem
