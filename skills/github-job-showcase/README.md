## 1. 项目概述

本仓库是一个 **Codex / Claude Code 双平台通用 Skill**，用于将 GitHub 项目包装为求职、面试、实习申请场景下的展示材料——README 重写、亮点提炼、简历 bullets 和面试话术。

## 2. 目标与场景

适用场景：

- 校招/实习简历的项目描述提炼。
- 面试中的项目介绍话术准备。
- GitHub README 对外展示优化。
- 课程设计/毕业设计项目的亮点包装。
- 技术 portfolio 的对外呈现。

## 3. 核心能力

- 基于仓库实际证据提取项目亮点。
- README 结构与对外展示优化。
- 简历项目描述（3-6 条）。
- 面试话术与项目定位陈述。
- 诚实包装——不夸大，只升级清晰度。

## 4. Quick Start

**Codex**：在 skill 市场中搜索 `github-job-showcase` 一键安装。

**Claude Code**：将本仓库克隆到 skills 目录或通过 plugin 加载：

    git clone https://github.com/however-yir/codex-skill-github-job-showcase.git
