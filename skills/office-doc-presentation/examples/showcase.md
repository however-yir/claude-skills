# Showcase Examples

These examples show how the skill packages messy information into polished, office-style deliverables that are actually ready to share.

## Example 1: Interview Project Deck

**Prompt**

`Turn my recruitment recommendation system notes into a 10-slide interview presentation with a clear story.`

**A strong output should include**

- a slide-by-slide storyline
- one message per page
- suggested diagrams or screenshots
- concise language suitable for presenting aloud

**Why this showcases the skill**

It demonstrates that the skill can move from raw project details to a believable, presenter-friendly deck.

## Example 2: PDF to Executive Brief

**Prompt**

`Summarize this long PDF into a concise manager-ready brief I can export to Word or WPS.`

**A strong output should include**

- a clean document structure
- key findings separated from background detail
- language that scans quickly
- assumptions called out when the source is ambiguous

**Why this showcases the skill**

This shows strong transformation ability, not just formatting cleanup.

## Example 3: Meeting Notes to Formal Follow-up

**Prompt**

`Convert these messy meeting notes into a polished follow-up document with decisions, action items, and next steps.`

**A strong output should include**

- clear decision summary
- ownership and next-step breakdown
- cleaned wording and section hierarchy
- a format that feels ready to send without manual rescue work

**Why this showcases the skill**

It highlights the skill's practical value for office-style communication and real work output.
